# beautiful-code

Example Clojure implementation of `eval-expr` based on William Byrd's [The Most Beautiful Program Ever Written]( http://paperswelove.org/2017/video/will-byrd-most-beautiful-program/).

## Usage

```bash
git clone
cd beautiful-code
lein repl
# open beautiful-code.core
# play with eval-expr
```
